# Fernando Montiel – Assignment Portfolio

## Project Name
Assignment Portfolio Website

## Description
This project is a personal portfolio hub built using HTML and CSS.
It serves as a centralized website that organizes and links all of my
web-based class assignments and projects.

The portfolio is designed to be simple, visually clean, responsive
across devices, and easy to extend as new assignments are completed.
Each assignment is accessible through a live HTTPS link hosted on
Microsoft Azure or GitHub Pages.

---

## Technologies Used
- HTML5
- CSS3
- Microsoft Azure Static Web Apps
- GitHub

---

## Compilation and Execution Instructions
This project is a static website and does not require compilation.

To run the project locally:
1. Download or clone the repository.
2. Open the `index.html` file in a modern web browser.

To view the live version:
- Open the Microsoft Azure HTTPS deployment link associated with this project.
